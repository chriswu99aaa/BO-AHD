## 四、详细研究方案

### 任务1：数学表述修正（对应 W6）

**问题**: 
1. 最大化/最小化符号约定不一致（论文中为 $\max f(h)$，但 TSP 问题最小化）
2. EI 公式与标准形式不符
3. $\beta$ vs $k$ 符号使用不一致

**解决方案**:
1. **统一目标函数**: 采用 $\tilde{f}(h) = -f(h)$ 转换，所有 BO 操作基于 $\tilde{f}(h)$ 最大化
2. **修正 EI 公式**: 使用标准 Expected Improvement 表达式
   $$ \text{EI}(h) = \sigma(h) [Z \Phi(Z) + \phi(Z)] $$
   其中 $Z = \frac{\mu(h) - f_{\text{best}} - \xi}{\sigma(h)}$
3. **统一符号**: 选择 $\beta$ 或 $k$，在全文保持一致

**实现细节**:
- 修改 `bo.py` 中所有相关计算，确保一致使用 $\tilde{f}(h) = -f(h)$
- 更新论文中所有公式表述
- 添加符号表，说明所有符号含义

### 任务2：KP 结果核查（对应 W8）

**问题**: KP 问题中报告超过"Optimal"的结果（OR-Tools 求解）

**解决方案**:
1. **验证 OR-Tools 设置**: 检查是否为精确求解器配置
2. **重新评估**: 使用相同实例重新运行 OR-Tools
3. **澄清表述**: 
   - 如果 OR-Tools 使用近似算法，注明为"近似最优"
   - 如果结果为精确解，检查评估指标的一致性

```python
def record_learning_curve(self):
    """记录学习曲线数据"""
    # 在每次评估后记录
    self.learning_curve.append({
        'eval_count': len(self.Y),
        'best_so_far': min(self.Y),
        'current': self.Y[-1],
        'time_elapsed': time.time() - self.start_time
    })
```

### A.2 成本监控实现

**文件**: `source/bo.py` (添加新方法)
```python
class CostMonitor:
    def __init__(self):
        self.start_time = time.time()
        self.llm_calls = 0
        self.evaluations = 0
        self.gp_fits = 0
        
    def record_llm_call(self):
        self.llm_calls += 1
        
    def record_evaluation(self):
        self.evaluations += 1
        
    def record_gp_fit(self):
        self.gp_fits += 1
        
    def get_summary(self):
        return {
            'total_time': time.time() - self.start_time,
            'llm_calls': self.llm_calls,
            'evaluations': self.evaluations,
            'gp_fits': self.gp_fits,
            'time_per_eval': (time.time() - self.start_time) / max(1, self.evaluations)
        }
```
